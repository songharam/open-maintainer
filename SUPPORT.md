# Support

Open Maintainer Workbench is currently a demo project for the Codex open source support program.

## Best Places to Start

- Bugs: open a bug report issue.
- Feature ideas: open a feature request issue and describe the maintainer workflow it improves.
- Documentation changes: open a docs issue.
- Questions: open a question issue.

## Before Opening an Issue

Please check:

- [README.md](./README.md)
- [docs/demo-script.md](./docs/demo-script.md)
- [docs/roadmap.md](./docs/roadmap.md)
- [docs/architecture.md](./docs/architecture.md)

## Scope

This project focuses on open source maintainer workflows. Requests for user accounts, databases, unrelated project management tools, or heavy backend systems are outside the current demo scope.
